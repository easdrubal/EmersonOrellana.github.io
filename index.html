Sobre Mí

Desarrollador Full Stack apasionado por la tecnología

Soy un desarrollador full stack certificado en JavaScript y Python con experiencia en la creación de aplicaciones web completas, desde el diseño de interfaces de usuario hasta la implementación de bases de datos y servidores.

Mi stack tecnológico principal incluye React.js y Vue.js para el frontend, Node.js y Django para el backend, y MongoDB y PostgreSQL para bases de datos. También tengo experiencia en DevOps con Docker y AWS.

Me encanta aprender nuevas tecnologías y enfrentar desafíos que me permitan crecer profesionalmente. Cuando no estoy codificando, disfruto contribuir a proyectos de código abierto y compartir conocimientos en blogs técnicos.
